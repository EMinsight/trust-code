


\textbf{Dt\_post}\index{Dt\_post}: This keyword is used to set the calculated statistics write period.

\textit{dts}: frequency value.

\textbf{t\_deb} value: Start of integration time

\textbf{t\_fin} value: End of integration time


\bigskip

\textit{stat}: Set to \textbf{Moyenne}\index{Moyenne}\textbf{ (average)} to calculate the average of the field
\textit{nom\_champ} (field name) over time or \textbf{Ecart\_type}\index{type}\index{Ecart\_type}\textbf{
(std\_deviation)} to calculate the standard deviation (statistic rms) of the field \textit{nom\_champ (field\_name)} or
\textbf{Correlation}\index{Correlation} to calculate the correlation between the two fields \textit{nom\_champ
}and\textit{ second\_nom\_champ.}


\bigskip

\textit{nom\_champ:} name of the field on which statistical analysis will be performed. Possible keywords are
\textbf{Vitesse}\index{Vitesse}\textbf{ (speed)}, \textbf{Pression}\index{Pression}\textbf{ (pressure)},
\textbf{Temperature}\index{Temperature}, \textbf{Concentration}\index{Concentration}\textbf{,{\dots}}


\bigskip

\textit{localisation}: localisation of post-processed field values (\textbf{elem} or \textbf{som}).


\bigskip
