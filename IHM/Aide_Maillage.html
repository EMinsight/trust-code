 TRUST mesh file example  



 Click on icon to maximize 
To enclose a file exemple.geom in a TRUST data file, 
you will use following method :

domaine ENVEL
Lire_fichier ENVEL exemple.geom

 To visualise this mesh,
 you have to transform it into a .lml file, 
 using following instructions.
 Then start up TRUST and post-process the file exemple.lml :


Ecrire_fic_lml ENVEL exemple.lml
