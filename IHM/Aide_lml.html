 TRUST results file .lml description  




Format of the ASCCI file nom_du_cas.lml :
(Only if the binary option was not asked for the post-processing)



nom_code		Characters string: The software name used
version             	Characters string: Software version
date               	Date interpreted in two-digit pairs as follows:
                        jj,mm,aa             (day, month, year )
			
nom_probleme   		Problem to be post-processed
			(characters string without space)  
comment         	Comment (without space)

format         		Key word FORMAT or BINAIRE 
 
GRILLE		Key word  

nom_grille          	Grid name (characters string)
dim_grille          	Integer: problem size  (2D or 3D)
nb_noeud            	Integer: grid nodes number 

xi yi  zi           	Nodes coordinates (from i=1 to nb_noeud)

TOPOLOGIE	Key word  

nom_topologie       	Characters string to describe the topology 
nom_grille         	Grid name relative to this topology

MAILLE		Key word

nb_maille           	Integer: meshes number

     for each mesh :
type_maille         	Element type (characters string) :
                      	Surfacic elements  :  POLY4 a POLY8
                      	Volumic elements   :  TETRA4
                                  		        PRISM6
		                              		VOXEL8  
   and
   
ie1  ie2 .. ien     	integers : mesh nodes list
FACE             	Key word

nb_face          	integer  : faces number 

           for each face :
type_face        	Face type (characters string) :
                   	Linear faces    :   LINE2
		   	Surfacic faces  :   POLY3 a POLY8
  et
  
if1 if2 .. ifm   	Face nodes list 
je1  je2         	The list of elements adjacent to the face 

TEMPS		Key word (necessary for each time step)
val_temps         	Time value at the  current time step  (A VERIFIER MERCI)

CHAMPPOINT	Key word

nom_champ         	Field name (characters string) 
nom_topologie     	The name of the topology where the field is defined by points                 
temps             	Time value

nom_var           	Name of the variable of the field
nb_comp           	Number of component(s) of the field
unite             	Unit of the variable (string of characters)
type_var          	Type of the discretisation of the variable (P1,P2 ..)
                  		(string of characters)
nb_points         	Number of points

n³ noeud et valeur du champ    List of values (for i=1,nb_points)

CHAMPFACE       	Key Word

nom_champ         	Name of the field (characters string) 
nom_topologie     	Name of the topology where the field by faces is defined
temps             	Time value 

nom_var           	Name of the  variable of the field
nb_comp           	Number of component(s) of the field
unite             	Unit of the variable (characters string)
type_var          	Type of discretisation of the variable (P1, P2 ..)
                  		(string of characters)
nb_faces          	Number of faces where the field is defined

n³ face et valeur du champ   list of data  (i = 1,nb_faces)

CHAMPMAILLE  	Key word

nom_champ         	Name of the field (characters string) 
nom_topologie     	Name of the topology where the field is defined by mesh
temps             	Time value

nom_var           	Name of the variable (characters string) 
nb_comp           	Number of component(s) of the field
unite             	Unit of the variable (characters string)
type_var          	Type of discretisation of the variable (P1, P2 ..)
                  		(string of characters)
                  		
nb_mailles       	Number of meshes where the field is defined

n³ maille et valeur du champ   Data list (for i = 1,nb_mailles )

FIN		Key word to close the post-processing file
