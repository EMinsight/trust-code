 Exemple de fichier de maillage de TRUST 



 Cliquer sur l'image pour l'agrandir

Si le maillage est contenu dans un fichier exemple.geom, pour
le relire dans un fichier de donnees TRUST il faudra utiliser
les instructions suivantes :

domaine ENVEL
Lire_fichier ENVEL exemple.geom

Pour eventuellement visualiser ce maillage en le transformant,
en un fichier de type lml, utiliser les instructions suivantes,
puis lancer TRUST et postraiter le fichier exemple.lml :

Ecrire_fic_lml ENVEL exemple.lml
